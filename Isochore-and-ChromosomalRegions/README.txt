This folder contains the scripts used for isochore and chromosomal region analyses in the associated publication https://doi.org/10.1093/gbe/evac090

Scripts were used as follows:
Wilcox-ExtractIsochores(FalcoRef8).bash extracted 'isochores', 100kb windows and compositional information, from all genomes
Wilcox-ExtractMegabaseIsochores(FalcoRef8).bash extracted megabase isochores, 1MB windows and compositional information, from all genomes
Wilcox-ChromosomeCompareOtherGenomes(FalcoRef8).bash downloaded other Inopinaves genomes for comparative analyses and annotations
Wilcox-MUMmerChromosomeRegionAnnotation(FalcoRef8).bash aligned falcon genomes to other Inopinaves with MUMmer for annotation past chromosome types by region
Wilcox-MUMmerFalcoChromosomeRegionAnnotation(FalcoRef8).bash aligned falcon genomes to falcons with MUMmer for annotation current chromosome types by region
Wilcox-ultChromAnnotateSummarizeIsochores(FalcoRef8).bash compiled and summarized information on isochores (100KB windows)
Wilcox-MergeIsochoreData(FalcoRef8).bash compiled and summarized information on isochores (1 MB)
Wilcox-OtherGenomesFullIsochores(FalcoRef8).bash annotated and summarized isochore compositional information for other amniotes
Wilcox-ultRevisedChromAnnotateSummarizeIsochores(FalcoRef8).bash annotated isochores by past and currect chromosome type using MUMmer alignments
