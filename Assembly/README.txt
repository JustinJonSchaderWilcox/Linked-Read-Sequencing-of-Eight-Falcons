This folder contains the scripts used for the extraction of de novo assembly of the linked reads used in this analysis https://doi.org/10.1093/gbe/evac090

Scripts were used as follows:
Wilcox-SupernovaOut(FalcoRef8).bash was used to extract several outputs for six of the falcons
Wilcox-PhasedSupernovaOutBlackShaheenPH2(FalcoRef8).bash was used to extract phased assmeblies from the Black Shaheen falcon
Wilcox-PhasedSupernovaOutKestrelPH2(FalcoRef8).bash was used to extract phased assmeblies from the common kestrel
